Գրել ծրագիր, որը կառավարում է գրադարանի գրքերի տվյալները։

Ծրագիրը օգտագործում է LibraryManager դասը և աշխատում է հրամանների միջոցով։ Բոլոր տվյալները կարդում է library_data.json ֆայլից, իսկ ավարտին պահպանում է տվյալները նույն ֆայլում։

Ծրագիրը պետք է աջակցի հետևյալ հրամանները․
add – ավելացնել գիրք (վերնագիր, հեղինակ, ժանր, հրատարակման տարեթիվ)
remove – հեռացնել գիրքը ըստ վերնագրի
search – որոնել գրքեր ըստ տեքստի
exit – պահպանել տվյալները և ավարտել ծրագիրը

Ծրագիրը աշխատում է անվերջ ցիկլում և սպասում է օգտատիրոջ մուտքագրված հրամաններին։
